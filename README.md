# public-trash
Android APP 垃圾分类识别
Kotlin+JetPack Compose+Material Design3

MVVM

Room

Retrofit+okhttp

1.注册登录功能

2.输入垃圾名称查询垃圾的分类，拍照识别垃圾的名称和分类

3.四大垃圾分类的概念介绍和图例

4.扔垃圾可以统计，做成饼图展示

5.可以更改头像，修改密码等
